从现在开始，接下来的正式定时任务统一按以下规则执行：

1. 执行主体固定为 `Codex`
2. 所有网页采集默认复用用户当前已登录的 `Chrome`
3. 微信采集固定使用：
   - `https://localhost:3001/`
4. 不再把 `Claude Desktop / OpenClaw / 浏览器插件 / 外接 MCP` 作为正式采集主执行器
5. 只允许复用以下已登录地址，不得新开未登录用户的浏览器窗口：
   - `https://www.jd.com/`
   - `https://localhost:3001/`
   - `https://shop.lenovo.com.cn/`
   - `https://www.tmall.com/`
   - `https://retail-pos.lenovo.com/`
6. 不允许为了采集信息再打开新的未登录 Chrome 窗口、访客窗口、无痕窗口或其它浏览器

## 目录规则

每个任务目录固定包含：

- `README.md`：该任务的执行口径、原始输入、命令入口、回执要求
- `原始输入/`：当天人工下载或页面保存的原始文件、截图、文本
- `运行回执/`：当天执行完成或未收口的回执
- `阻塞记录/`：登录失效、页面白屏、缺原始输入、权限弹窗等阻塞记录

## 统一执行规则

所有任务统一遵守：

1. 操作前先扫描当前状态
2. 只有目标页稳定可操作时，才允许执行一次手工单步动作
3. 每执行一步后，必须再次扫描状态
4. 出现登录页、二维码、白屏、403、安全验证、错误弹层、权限弹窗，立即停止并写阻塞
5. 不得把旧文件重跑写成当日实时采集
6. 不得把脚本跑通写成业务已完成

固定动作门禁提示词：

```text
操作前先扫描目标当前状态，确认当前窗口/页面、登录态、加载态、目标控件是否正确；
只有目标页稳定可操作时，才允许执行一次手工单步动作；
每执行一步后，必须再次扫描状态，再决定下一步；
出现登录页、二维码、白屏、转圈超时、403、安全验证、错误弹层、底部菜单栏收缩或非目标页，立即停止并写阻塞；
禁止连点、双击、盲点、错误状态下硬操作。
```
