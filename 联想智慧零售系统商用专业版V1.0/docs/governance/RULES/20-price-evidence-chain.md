# 价格证据链规范

## 适用范围

门店零售价、京东价、联想官旗价、天猫/淘宝备用价、群报价、灰渠公众号报价、竞品报价、国补价、教育补贴、营销 PO。

## 强制规则

- 价格必须有来源平台、采集时间、入口链接、截图或原始文件。
- 平台价格只能作为价格信号，不能直接覆盖门店零售价。
- 历史报价沿用必须标记 carry_forward。
- 活动金额为 0 或缺证据时，不得写成已完成。
- 下架、403、安全验证、验证码时记录阻塞，不补假价。

## 写入链路

```text
price_raw_evidence -> price_signal -> retail_db -> API -> frontend
```
