# TASK_ANALYSIS

## 任务

生成独立软件包：`联想智慧零售系统商用专业版V1.0`。

## 输入

- 最终规范包：`Lenovo_SmartRetail_Governance_V5_Enterprise.zip`
- 项目规则：真实数据优先、证据优先、SN 优先、可审计、可追溯

## 目标

建立一个可继续商业化开发的新系统目录，包含前端界面、数据结构、治理规范、验收文档，并按原版零售系统菜单顺序接入现有 SQL/API 真实数据。

## 本轮不做

- 不覆盖现有 `apps/web-cockpit` 主驾驶舱。
- 不冒充真实智店通、微信群、京东、公众号采集结果。
- 不写入真实门店价格、库存、SN 或账号凭证。

## 风险

当前已建立 API 桥接，但生产验收仍需要逐个子页面做真实页面可见验收，并确认每条业务数据有 SQL/API/证据链。
