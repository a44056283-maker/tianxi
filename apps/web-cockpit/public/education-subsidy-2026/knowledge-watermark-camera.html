"database is locked" **原因**：SQLite 写锁竞争。 **解决**：API server 内部已用单连接 (commit 2026-06-09)，仍出现时可手动： ```bash python3 -c "import sqlite3; c=sqlite3.connect('apps/api-server/data/retail-core.sqlite3', timeout=20); c.execute('PRAGMA wal_checkpoint(TRUNCATE)'); c.close()" ``` ### 2. "OCR call failed" **原因**：OCR 服务 (port 8765) 未启动。 **解决**： ```bash launchctl load -w ~/Library/LaunchAgents/com.lenovo-smart-retail.ocr-service.plist curl -s http://127.0.0.1:8765/health ``` ### 3. "submit record failed" **原因**：API server (port 8000) 异常。 **解决**： ```bash launchctl unload ~/Library/LaunchAgents/com.lenovo-smart-retail.api-server.plist launchctl load -w ~/Library/LaunchAgents/com.lenovo-smart-retail.api-server.plist curl -s http://127.0.0.1:8000/api/education-scan/v2/stats ``` ### 4. 文件没有被识别 **检查文件名格式**：必须含 `EMP003-006` 之一，或中文员工名。 **手动验证**： ```bash python3 -c " from pathlib import Path import sys; sys.path.insert(0, 'scripts') from watermark_camera_sync import extract_metadata_from_filename print(extract_metadata_from_filename('EMP003_梁伟_三件套_xxx.png')) " ``` ## 验收标准 - ✅ 拍照 → 落地到 `incoming/` → 自动入库 → 移动到 `processed/` - ✅ 失败 → 移动到 `failed/`，保留可重试 - ✅ OCR 识别成功率 > 80% - ✅ 匹配成功 → 自动锁定 8 字段（订单/客户/SN/SKU/产品/PN/规格/折扣） - ✅ 性能归属自动计算（店长 20%/服务顾问 80%） - ✅ sync_report_{date}.json 全链路审计 - ✅ launchd 守护进程自动运行 - ✅ 失败重试（最多 3 次，指数退避 2/4/8 秒） ## 版本历史 - **v1.0 (2026-06-09)**: 初版，支持单扫/多扫/二件套/三件套，4 名员工，自动 OCR + SQL 同步
