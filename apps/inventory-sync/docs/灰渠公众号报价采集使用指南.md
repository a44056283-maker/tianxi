# 灰渠公众号报价采集使用指南

## 功能概述

灰渠公众号报价采集模块用于通过 Chrome `https://localhost:3001/` 已登录网页微信，采集微信公众号「郑州市创业」的市场批发报价。当前唯一允许的公众号入口是：`文件传输助手` 聊天记录区下面用户固定放置的公众号入口。

## 当前固定流程

1. 复用用户默认 Chrome 已登录会话，打开或切回 `https://localhost:3001/` 网页微信。
2. 进入 `文件传输助手` 聊天记录区。
3. 点击聊天记录下面用户固定放置的「郑州市创业」公众号入口。
4. 进入公众号后，必须点击页面最下面带日期的报价快捷入口，日期必须为当天或当前最新有效报价日期。
5. 只认可该快捷入口打开后的报价页、当天截图或 OCR 证据。
6. 保存 raw 文本、截图或 OCR 整理文本到 `artifacts/manual/gray-wholesale-YYYY-MM-DD.txt` 或 `.md`。
7. 执行同步任务，把报价写入 SQL/API 主链并刷新前端。

```bash
cd apps/inventory-sync
npm run run:scheduled-task -- daily-gray-channel-check
```

## 禁止流程

- 禁止打开微信桌面版。
- 禁止新开浏览器/Profile。
- 禁止使用 Browser/in-app browser/browser-use/Playwright/Puppeteer/Chromium launch 打开外部采集页。
- 禁止公众号名称搜索。
- 禁止收藏夹、文章列表、历史原文、旧截图或旧文件重跑冒充当天采集。
- 进入公众号页本身不算完成；未点击最下面带日期的报价快捷入口，不得解析或收口。

## 执行流程

```text
Chrome 已登录网页微信
-> 文件传输助手
-> 聊天记录下面固定公众号入口
-> 公众号页最下面带日期的报价快捷入口
-> 保存 raw 文本/截图/OCR
-> daily-gray-channel-check
-> SQL/API 报价主链
-> 前端库存/报价页面可见验收
```

## 阻塞口径

- 最下面带日期的报价快捷入口不存在：`blocked_missing_input`。
- 日期不是当天或当前最新有效报价日期：`blocked_missing_input`，可沿用旧值但必须标记 carry forward。
- 入口打不开、网页微信掉线、白屏、登录页、验证码、安全验证：`blocked_page_risk`。
- 只有历史原文、旧截图或旧文件：不得写 `real_completed`。

## 验证

任务完成必须同时满足：

1. 有本轮 raw 文本、截图或 OCR 证据。
2. `latest-gray-wholesale-quotes.json` 或 SQL 报价主链已更新。
3. 前端对应报价区域可见本轮结果。
4. 任务报告写明 `real_completed`，否则按阻塞或未收口状态记录。
