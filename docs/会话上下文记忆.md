  `180 天`
- 判断口径：
  - 以 SN 的 `inboundDate` 计算 `stockAgeDays`
  - `stockAgeDays >= 180` 视为超期库存
- 当前系统要求：
  - 前端展示 `陈旧库存提醒`
  - 零售与库存管理后续定时任务按该阈值巡检
  - 提醒店面优先销售超期库存
