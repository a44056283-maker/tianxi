# 库存商品京东式分类规则

## 目标

库存展示不直接沿用智店通的原始分类，而是按京东店铺导购目录整理，便于后续接京东价格、联想官网价格、国补价格和电子价签。

## 分类优先级

1. 优先使用 `联想京东自营旗舰店目录`。
2. 联想自营目录没有或覆盖不准确的商品，按对应官方店或品牌线补充目录。
3. 智店通原始分类保留在 `sourceCategory` 字段，只作为追溯依据。

## 来酷 / 斗战者补充规则

来酷、斗战者系列如果联想自营目录没有完全覆盖，则标记为 `来酷京东官方目录补充`。

|---|---|---|
