# OpenClaw 训练场景模板库

更新时间：2026-05-17

## 总流程

以后所有采集任务、定时任务、手工复核任务都必须按下面 4 步走：

1. Codex 先做演示
2. OpenClaw 练习单能力模板
3. OpenClaw 练习场景自定义任务
4. Codex 验收通过后，才下达真实任务

不能跳过前 3 步直接给正式任务。

## 第一层：单能力模板

### 模板1：进页

适合场景：

- 智店通页面进入
- 京东详情页打开
- 公众号文章页进入

验收标准：

1. 正确进入
2. 正确识别页面状态
3. 截图
4. 写回执

### 模板2：命中控件

适合场景：

- 日期框
- 下拉框
- 导出按钮
- 搜索按钮

验收标准：

1. 命中正确控件
2. 判断是否触发
3. 截图前后变化
4. 记录偏移建议

### 模板3：导出动作

适合场景：

- 智店通导出
- 群文件下载
- 公众号原图保存

验收标准：

1. 真的触发下载或保存
2. 记录文件名
3. 记录路径

### 模板4：内容提取

适合场景：

- Excel 导出解析
- OCR 提取
- 页面文本结构化

验收标准：

1. 指明来源
2. 指明关键字段
3. 失败时明确原因

### 模板5：分类归档

适合场景：

- 群报价库
- 公众号报价库
- 竞品库
- 智店通证据库

验收标准：

1. 按日期归档
2. 按来源归档
3. 按产品分类 / 系列 / 型号归档
4. receipt 可复核

## 第二层：场景自定义任务

单能力通过后，才能下达场景组合任务。

### 场景A：智店通导出链

顺序：

1. 进页
2. 命中日期框
3. 命中筛选框
4. 命中导出按钮
5. 确认文件落点
6. 解析导出内容

### 场景B：微信群报价链

顺序：

1. 进入群
2. 找当天报价
3. 下载文件或截图
4. 确认文件落点
5. OCR / 结构化
6. 分类归档

### 场景C：公众号报价链

顺序：

1. 进入文章
2. 找当天有效报价
3. 保存原文或截图
4. OCR / 文本提取
5. 按系列型号归档

### 场景D：京东 / 联想 / 竞品链

顺序：

1. 打开详情页
2. 确认商品一致
3. 截图价格与配置
4. 提取字段
5. 分类归档

## 第三层：正式任务准入

必须同时满足：

1. 对应单能力模板练通
2. 对应场景组合任务练通
3. receipt 自动进入待 Codex 接收
4. Codex 验收通过

只有这样，才能进入：

1. 正式采集任务
2. 正式定时任务

## Receipt 触发规则

OpenClaw 完成一轮后，receipt 不能只是落在文件夹里。

必须自动触发：

1. 进入会话看板消息流
2. 进入 `待 Codex` 队列
3. 由 Codex 审核后决定：
   - 继续练习
   - 下达修复任务
   - 进入正式任务
