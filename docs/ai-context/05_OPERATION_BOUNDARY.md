# 05 Operation Boundary

更新时间：2026-05-12

## 核心边界

- 只能在本项目目录内创建、修改代码和文档。
- 不操作真实账号的登录、支付、删除、提交、权限变更。
- 不读取、不保存密码、Token、Cookie、验证码、会话文件。
- 需要第三方业务数据时，优先：
  - 用户导出 Excel / CSV
  - 用户提供截图
  - 用户已打开页面后，仅读取可见信息

## 可做

- 解析本地文件
- 修改前后端代码
- 构建快照和本地 API
- 打开本地网页审计 UI
- 在用户已打开的第三方页面中读取可见价格/字段
- 通过人工辅助流程把可见信息写回本地快照

## 不可做

- 代替用户登录第三方真实系统
- 保存或导出第三方会话状态
- 处理验证码、短信、滑块、人机验证
- 自动执行外部可见的提交、支付、删除、权限变更
- 绕过风控

## 风控停止条件

出现以下任一情况，必须立即停止当前路径：

- `403`
- 验证码
- 滑块
- 登录验证
- 安全提示
- 页面异常跳转
- 访问受限

## 外部网页采集硬边界

- 京东、联想商城、天猫/淘宝、智店通、网页微信、联想保修页、竞品排行等外部页面，禁止用脚本、无头浏览器、browser-use、Browser/in-app browser、Playwright/Puppeteer/Chromium launch 或高频自动点击采集。
- 外部网页真实采集只能复用用户当前已经登录的默认 Chrome 可见窗口，通过低频手动打开、手动点击、手动读取可见标题/规格/价格。
- 禁止重新打开新的浏览器、打开空白浏览器、新建 Chrome Profile、清理登录缓存、主动退出账号或用无登录态窗口替代已有会话；智店通若被动退出或跳回登录页，只允许在同一默认 Chrome 会话中按固定登录恢复链尝试一次。
- 本地脚本只能用于项目内数据检索、人工证据写回、快照重建、前端生成和播报，不能承担外部网页采集动作。
- 如果没有可见 Chrome 手动证据，相关字段只能标注为待采、线索或阻塞，不能写成 confirmed/captured。

## 数据写入边界

- 未锁定规则、未校准匹配的数据，不得进入正式报价引擎
- 未确认的第三方价格可以作为线索，但必须标注状态
- 销售出库、采购入库、其他出库要区分业务类型，不能混写

## ERP / 后台 UI 边界

- 订单、库存、SN、台账、审计、出入库、价保、实物仓等 ERP / 后台类页面，禁止使用大方块卡片式占位作为主布局。
- 这类页面的默认主结构必须是：
  - 工具条
  - 同层筛选栏
  - 紧凑摘要条
  - 表格 / 台账
  - 明细 / 审计 / 作业区
- 禁止以下做法：
  - 用多张大卡片堆 KPI 再把台账挤到页面下方
  - 用营销英雄卡、驾驶舱卡片或大圆角信息块代替订单/库存主表
  - 用大量说明文字占位，而不是直接给字段、状态、数量和动作位
- 只有面向客户的营销展示页、广告机页、零售英雄卡页，才允许继续使用客户可见的卡片化表达；后台和 ERP 页不适用这一套。

## 远程域名访问边界

- 前端不能只按局域网 IP 或本机 `127.0.0.1` 适配；后续开发必须同时考虑远程域名、手机端和平板端访问。
- 远程域名访问不得默认拼接 `当前域名:8000` 作为唯一数据源；这会在 HTTPS、反向代理或 8000 未公开时导致数据区白屏。
- 标准数据链路固定为：
  - 本机开发：`127.0.0.1:5174 -> 127.0.0.1:8000`
  - 局域网 / 远程域名：优先同源 `/api`，由前端服务或正式网关代理到 `8000`
  - API 不通时：立即回落到 `/data/*.json` 静态快照，只读页面不能整块空白
- 写操作仍必须走 API；远程只读展示可以使用静态快照兜底，但必须明确“编辑需回到本机 API 模式”。
- 任何新增前端模块都必须同时提供 API 读取和 `/data` 静态快照兜底，不能只在本机 API 模式下可见。
- 如果使用 Vite dev/preview 承载远程域名，必须确认 Host 头不会被拦截；域名入口返回 `403` 时，优先检查 `allowedHosts` 或正式网关 Host 透传配置。

## 汇报边界

- 禁止打嘴炮。
- 禁止未执行先汇报。
- 禁止用推测替代实操结果。
- 禁止用“估计”“大概”“应该”描述命中率、采集完成度、页面命中情况。
- 所有这类结论必须能回溯到：
  - 实际操作
  - 实际截图
  - 实际 OCR
  - 实际练习台账
## 2026-05-16 OpenClaw 使用边界

- OpenClaw 只允许操作证据采集层。
- 允许写入：
  - `apps/inventory-sync/artifacts/manual/*`
- 禁止写入：
  - 任意正式 `latest-*.json`
  - 任意前端直接消费快照
- OpenClaw 发现掉线、验证码、403、白屏后必须立即停止并返回主线程，不得自行绕过。
