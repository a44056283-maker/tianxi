## 单步操作规则

每次只允许做一个动作：

1. 截图当前页
2. 判断当前页是否还是 `库存订单`
3. 只点击一个目标坐标
4. 等待页面变化
5. 再截图
6. 判断是否触发成功

禁止：

1. 连点
2. 同一轮里同时试多个控件
3. 因为失败就跳去别的页面
4. 用脚本补点

## 训练顺序

### 第1轮：只练 `交易日期`

只做：

1. 进入 `库存订单`
2. 点击 `交易日期` 输入框中心：`(692, 289)`
3. 如果无变化，再下一轮点击 `日历图标`：`(788, 289)`

成功判定：

1. 日期选择器展开
2. 或输入框获得明显焦点状态

失败判定：

1. 点击后完全无变化
2. 页面状态不变
3. 焦点未切换

### 第2轮：只练 `入出库类型`

只做：

1. 进入 `库存订单`
2. 点击 `入出库类型` 输入框中心：`(990, 239)`

成功判定：

1. 下拉展开
2. 或输入框获得焦点

失败判定：

1. 没有下拉
2. 没有焦点
3. 仍然只是占位文字

### 第3轮：只练 `导出`

前提：

1. 页面已确认可操作
2. 至少 `搜索` 或筛选区有一个动作已成功

只做：

1. 点击 `导出` 按钮：`(387, 339)`
2. 观察是否触发下载或导出提示
3. 确认是否生成文件
