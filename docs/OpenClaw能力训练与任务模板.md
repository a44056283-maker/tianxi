# OpenClaw 能力训练与任务模板

更新时间：2026-05-17

## 目标

OpenClaw 不是直接接整条业务链，而是先训练单项能力，再把能力串起来。

我们让它采集这些东西，目的只有三个：

1. 留下真实证据
2. 形成可复核的结构化结果
3. 交给 Codex 审核后再同步前端

不是为了让它“看起来在点网页”。

## 能力地图

### 1. 页面进入

要回答：

- 能不能进页面
- 页面是不是对的
- 页面是不是卡住了

### 2. 入口定位

要回答：

- 导出按钮在哪里
- 日期框在哪里
- 筛选框在哪里
- 页签是不是点对了
- 这是网页控件还是纯桌面控件
- 应该用 DOM/控件定位，还是只能用桌面点击

#### 网页型页面的固定规则

如果目标是网页页面，不允许默认使用“截图坐标点控件”作为主链路。

必须先判断：

1. 这个控件能不能被浏览器识别
2. 这个输入框/步进器/按钮能不能被控件定位
3. 这个页面是不是应该优先走已登录浏览器会话的 DOM/控件链路

固定优先级：

1. DOM/控件定位
2. 文本 + 角色定位
3. 坐标辅助验证

只有纯桌面软件、或网页控件完全无法暴露时，才允许坐标点击成为主动作。

### 3. 导出执行

要回答：

- 能不能导出
- 导出的到底是什么
- 是下载文件还是截图保存

### 4. 文件落点确认

要回答：

- 文件保存到了哪里
- 文件名是什么
- 后续谁来读它

### 5. 内容提取

要回答：

- 页面文本能不能直接读
- 截图能不能 OCR
- Excel 能不能解析

### 6. 分类归档

要回答：

- 这条证据属于哪个业务
- 属于哪一天
- 属于哪个产品分类 / 系列 / 型号

## 任务模板

### 模板A：只练进页

```text
本轮只训练页面进入能力。
目标页面：...

只做：
1. 进入页面
2. 确认页面名称、URL、状态
3. 截图
4. 回执

不做：
1. 不点筛选框
2. 不点导出
3. 不提取数据
4. 不分类

成功标准：
1. 正确进入页面
2. 正确识别页面状态
3. 截图保存成功
```

### 模板B：只练导出

```text
本轮只训练导出能力。
目标页面：...
目标按钮：...

只做：
1. 进入目标页面
2. 找到导出按钮
3. 尝试导出
4. 确认是否生成文件
5. 写回执

不做：
1. 不解析导出内容
2. 不更新业务结果
3. 不进入其他页面

成功标准：
1. 点到正确导出按钮
2. 确认文件是否生成
3. 确认文件路径
```

### 模板C：只练提取与分类

```text
本轮只训练提取与分类能力。
输入来源：...

只做：
1. 读取已有文件 / 截图 / 页面文本
2. 提取关键字段
3. 按类别归档
4. 写回执

不做：
1. 不重新打开别的页面
2. 不重复导出
3. 不写正式快照

成功标准：
1. 字段提取准确
2. 分类结果清楚
3. 回执可被 Codex 读取
```

## 当前推荐训练顺序

## 统一前置门禁

无论训练哪一种采集能力，都必须先过同一个前置门禁：

1. 先扫描目标当前状态
2. 判断当前是否处于可操作状态
3. 只执行一次手工单步动作
4. 动作后再次扫描状态
5. 状态正确才允许下一步

固定检查项：

- 当前窗口/标签页是不是目标
- 当前是不是已登录
- 当前是不是已加载完成
- 当前有没有白屏、转圈、验证码、403、安全验证、错误弹层、菜单栏收缩、非目标页
- 当前目标控件是不是稳定可见

固定禁止项：

- 未扫状态就点
- 连点
- 双击补救
- 盲点正文区、文章区、表格区、底部菜单区
- 在错误状态下继续硬操作

### 智店通

1. 首页进入
2. 菜单命中
3. 日期框 / 筛选框 / 导出按钮命中
4. 导出动作
5. 下载目录确认
6. 导出文件解析

### 微信群 / 公众号

1. 进入来源页
2. 命中当天有效内容
3. 截图 / 原文保存
4. OCR / 文本提取
5. 产品分类 / 系列 / 型号归档

固定前置门槛：

1. 每一步动作前先扫当前状态
2. 只允许单步点击
3. 禁止连点、双击、盲点文章区
4. 进入登录页、二维码页、白屏、底部菜单栏收缩态时立即停止
5. 不允许把错误状态下的继续尝试当成“练习”

### 所有训练任务的固定提示词

```text
操作前先扫描目标当前状态，确认当前窗口/页面、登录态、加载态、目标控件是否正确。
只有目标页稳定可操作时，才允许执行一次手工单步动作。
每执行一步后，必须再次扫描状态，再决定下一步。
出现登录页、二维码、白屏、转圈超时、403、安全验证、错误弹层、底部菜单栏收缩或非目标页，立即停止并写阻塞。
禁止连点、双击、盲点、错误状态下硬操作。
```

### 京东 / 联想 / 竞品

1. 命中已锁定商品页
2. 确认商品一致
3. 截图价格与活动
4. 提取价格与配置
5. 按品牌和型号归档
