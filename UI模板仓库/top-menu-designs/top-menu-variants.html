方案 A

联想官旗式：品牌红 + 商品详情页签

适合商品零售、产品价保、报价来源等以商品为中心的场景。

Lenovo智慧零售
方案 B

京东自营式：高密度分类 + 状态数字

适合销售人员快速看库存、价格、缺口、待同步数量。

京东式经营导航
方案 C

Apple 极简式：低噪声横向入口

适合让界面更干净，但业务状态需要放到二级区域展示。

华为商城式：产品族分区 + 双动作

适合首页和商品零售，把浏览、查询、收银动作分开。
